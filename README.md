# LinkNews24

বাংলাদেশের পাঠকদের জন্য একটি ক্লিন, মিনিমাল ও প্রফেশনাল বাংলা অনলাইন নিউজ পোর্টাল।

## স্ট্যাটাস

বর্তমানে ফ্রন্টএন্ড ফাউন্ডেশন (ডিজাইন সিস্টেম + স্ট্যাটিক/মক ডেটা) নিয়ে কাজ চলছে। ব্যাকএন্ড/CMS পরবর্তী ধাপে যুক্ত হবে।

## মনোরিপো স্ট্রাকচার

```
apps/
  web/      → Next.js পাবলিক ফ্রন্টএন্ড (App Router, TypeScript, Tailwind CSS)
docs/       → PRD ও ডেভেলপমেন্ট রোডম্যাপ
```

## টেক স্ট্যাক

- ফ্রন্টএন্ড: Next.js (App Router) + TypeScript + Tailwind CSS
- ব্যাকএন্ড (আসছে): Node.js + Express, PostgreSQL, Prisma
- রিয়েল-টাইম (আসছে): Socket.io + Redis

## ডেভেলপমেন্ট

```bash
cd apps/web
npm install
npm run dev
```

## ডকুমেন্টেশন

বিস্তারিত প্রোডাক্ট রিকোয়ারমেন্ট ও ফেজ-ভিত্তিক ডেভেলপমেন্ট রোডম্যাপের জন্য [docs/](docs/) দেখুন।
