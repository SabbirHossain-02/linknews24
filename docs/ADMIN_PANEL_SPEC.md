# LinkNews24 — অ্যাডমিন প্যানেল স্পেসিফিকেশন (v1)

> কপি-পেস্ট রেডি স্পেক। থিম: navy (#14181f) + crimson (#d81f26), ফন্ট Siyam Rupali/Hind Siliguri, clean-minimal। নীতি: **সবকিছু ≤ ২ ক্লিকে**।

---

## ০. আর্কিটেকচার
- পাবলিক সাইট: **Vercel** (Next.js)
- অ্যাডমিন প্যানেল: Next.js `/admin` রুট (auth-guarded), VPS-এর API-তে কল
- ব্যাক-এন্ড: **VPS** — Node + Express + Prisma + PostgreSQL (DB `linknews24`, পোর্ট 4100), JWT (httpOnly cookie), PM2 `ln24-api`
- মিডিয়া: VPS local + Nginx (পরে S3-compatible), ছবি WebP অপটিমাইজড

## ৩. সেকশন বিস্তারিত

- স্ট্যাট কার্ড: আজকের ভিউ, মোট আর্টিকেল, pending ড্রাফট, নতুন কমেন্ট, নতুন সাবস্ক্রাইবার

- ট্রাফিক গ্রাফ (৭/৩০ দিন)

### 📝 আর্টিকেল
- **লিস্ট ভিউ**: থাম্বনেইল, টাইটেল, ক্যাটাগরি, লেখক, স্ট্যাটাস, তারিখ, ভিউ
  - ইনলাইন টগল: Publish/Draft, Breaking, Feature
  - ফিল্টার: ক্যাটাগরি / স্ট্যাটাস / লেখক / তারিখ; সার্চ; Bulk action
- **এডিটর পেজ** (এক পেজে সব):
  - টাইটেল (bn/en), স্লাগ (অটো), Excerpt
  - রিচ টেক্সট এডিটর (উপরের সব ফিচার)
  - ফিচার্ড ইমেজ আপলোড
  - ক্যাটাগরি (একাধিক), ট্যাগ
  - সাইড প্যানেল: **SEO** (meta title/description, keywords, OG image), **শিডিউল** (তারিখ+সময় অটো-পাবলিশ), **Breaking** টগল, লেখক, স্ট্যাটাস

## ৭. নিরাপত্তা
- JWT (httpOnly cookie) + bcrypt, রোল-বেইজড মিডলওয়্যার
- সব ইনপুট validation (Zod), rate-limiting
- CORS শুধু Vercel ডোমেইনে
- API HTTPS (nip.io/sslip.io + Let's Encrypt, ডোমেইন না থাকা পর্যন্ত)
- `.env` git-এ নয়; আলাদা DB ইউজার; নিয়মিত ব্যাকআপ
- pkg.net সম্পূর্ণ অস্পর্শিত (আলাদা পোর্ট/DB/Nginx block/PM2)
